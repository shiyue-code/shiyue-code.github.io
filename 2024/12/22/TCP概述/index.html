TCP

TCP是一种面向连接的可靠的数据传输。

1. TCP头部概述

  • Source Port （源端口）: 16位
  • Destination Port（目的端）: 16位
  • Sequence Number （序列号）：32位
  • Acknowledgment Number（确认序列号）：32位
  • Offset （偏移）：4位， data起始位置
  • Reserved（保留）：4位
  • TCP Flags（标志）：8位
    • CWR: 0x80 Congestion Window Reduced
    • ECE: 0x40 ECN Echo
    • URG: 0x20 Urgent
    • ACK: 0x10 Acknowledgment
    • PSH: 0x08 Push
    • RST: 0x04 Reset
    • SYN: 0x02 Synchronization
    • FIN : 0x01 Finished
  • Window（数据窗）： 16位
  • Checksum（CRC校验）：16位
  • Urgent Pointer（紧急指针 OOB ）：16位
  • TCP Options(variable length, optional) （选项）：？？位

2. TCP的可靠传输

  1. TCP把所有要发送的数据编号（每个字节一个编号），来记录当前发送的数据和确认收到的数据并且以此实现重发，即确认与重发机制。
  2. 发送的数据序列号和数据大小由TCP头部的Sequence Number和Window记录。
